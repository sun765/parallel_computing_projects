*************************How to excute:  *************************
1.run "make" to compile 
2.run "output" to run

************************Process of the program******************
0. Input the array size and the number of threads you want to use

1. First the program will create an array (arr) and fill it with random numbers, it will also create a deep copy of it (copy_arr)

2. Run the quicksort using only one thread. (just call the qsort function in the library on the copy_arr). After the sort is done, report the
time taken

3. Run the parallel quicksort using required number of threads on arr. After the sort is done, report the time taken;

4. Report the performance, and check correctness of the parallel quicksort


**********************How to change parameters***************

at the top of the main file , you can change the parameters .
